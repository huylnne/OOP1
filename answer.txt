 1. Lớp nào nên triển khai giao diện Comparable?
 Trả lời: Lớp mà bạn muốn so sánh đối tượng trong giỏ hàng cần triển khai giao diện Comparable. Trong trường hợp này, vì chúng ta muốn so sánh các đối tượng của lớp Media và các lớp con của nó (như DVD, Book, v.v.), nên lớp Media (một lớp trừu tượng) sẽ triển khai giao diện Comparable. Điều này đảm bảo rằng tất cả các lớp con của Media có thể được so sánh.

 2. Cách triển khai phương thức compareTo() để sắp xếp theo title và cost?
 Trả lời: Để sắp xếp các đối tượng theo title trước, rồi đến cost, phương thức compareTo() cần so sánh từng thuộc tính này theo thứ tự ưu tiên. Cách triển khai có thể như sau:
@Override
public int compareTo(Media other) {
    // So sánh theo title trước
    int titleComparison = this.getTitle().compareTo(other.getTitle());
    if (titleComparison != 0) {
        return titleComparison;
    }
    // Nếu title giống nhau, so sánh theo cost
    return Double.compare(this.getCost(), other.getCost());
}

// 3. Có thể có hai quy tắc sắp xếp (theo title và cost, và theo cost và title) trong giao diện Comparable không?
// Trả lời: Không thể có hai quy tắc sắp xếp khác nhau trong một giao diện Comparable. Phương thức compareTo() trong Comparable chỉ có thể định nghĩa một quy tắc sắp xếp duy nhất cho mỗi lớp. Vì vậy, nếu bạn cần nhiều quy tắc sắp xếp, bạn sẽ phải sử dụng Comparator, nơi có thể định nghĩa nhiều phương thức so sánh khác nhau.

 4. Làm thế nào để tạo quy tắc sắp xếp riêng cho DVD với độ dài giảm dần?
 Trả lời: Nếu các đối tượng DVD cần có quy tắc sắp xếp riêng (theo title, rồi đến length giảm dần, và cuối cùng là cost), bạn có thể tạo lớp DVD kế thừa từ Media và triển khai phương thức compareTo() riêng biệt cho lớp DVD. Đây là cách bạn có thể làm:
public class DVD extends Media implements Comparable<DVD> {
    @Override
    public int compareTo(DVD other) {
        // So sánh theo title trước
        int titleComparison = this.getTitle().compareTo(other.getTitle());
        if (titleComparison != 0) {
            return titleComparison;
        }
        
        // Nếu title giống nhau, so sánh theo length giảm dần
        int lengthComparison = Integer.compare(other.getLength(), this.getLength()); // Đảo ngược để sắp xếp giảm dần
        if (lengthComparison != 0) {
            return lengthComparison;
        }

        // Nếu title và length giống nhau, so sánh theo cost
        return Double.compare(this.getCost(), other.getCost());
    }
}
